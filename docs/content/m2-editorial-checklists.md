# M2 Editorial Checklists

## Event checklist

- title present in `it`, reviewed in `en`
- summary present in `it`, reviewed in `en`
- valid `status`
- `startDate` checked
- cover image chosen only if alt text is ready in both locales
- `featuredOnHome` used only for current priority events

## Homepage proof checklist

- quote is guest-safe and public-ready
- source is specific enough to be credible
- no placeholder quotes
- no quote without a matching source

## Media checklist

- `alt` completed in `it`
- `alt` reviewed in `en`
- usage area assigned
- readiness state assigned
- credit/source note filled when needed

## Bilingual readiness checklist

- Italian copy authored first
- English reviewed against meaning, not translated word-for-word
- slugs and route-level labels stay code-owned unless explicitly modeled
